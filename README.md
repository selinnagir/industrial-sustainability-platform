# Industrial Sustainability Analytics Platform

This project is an industrial sustainability analytics platform developed for energy and carbon performance monitoring.

## Features
- Data loading and preprocessing
- Missing value checks and type conversions
- KPI cards
- Trend analysis
- Sector benchmarking
- eGRID / state benchmark analysis
- Sustainability score
- Anomaly detection
- Forecasting
- AI chat module
- Automatic PDF reporting

## Tech Stack
- Backend: Python / FastAPI
- Frontend: React / TypeScript
- Reporting: ReportLab

## Run
Backend and frontend should be started separately according to project structure.

## Notes
Sensitive files such as `.env` are excluded from version control.
